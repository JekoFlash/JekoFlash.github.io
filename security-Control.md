# Security Control

![alt text](./assets/SecurityControl.jpg)

## Type Of Security Control

- Administrative Controls

## Control Function

- Preventative

## Location

- Swimming Pool at Kasetsart University Sriracha Campus

## Why is the swimming regulations sign considered Administrative Controls and Preventative?

- ป้ายกฎระเบียบการใช้สระว่ายน้ำถูกจัดเป็น Administrative Controls เพราะ เป็นมาตรการที่กำหนดขึ้นโดยฝ่ายบริหารหรือหน่วยงานที่ดูแลสถานที่ เพื่อให้แน่ใจว่าผู้ใช้ปฏิบัติตามกฎระเบียบที่กำหนดการแต่งกายที่เหมาะสม
และป้ายกฎระเบียบยังถือเป็น Preventative Control เพราะมีจุดประสงค์เพื่อป้องกันปัญหา เช่น การกำหนดให้ใส่ชุดว่ายน้ำที่ถูกต้องช่วยป้องกันการปนเปื้อนของน้ำในสระ, การจำกัดไม่ให้เด็กลงสระโดยไม่มีผู้ปกครองช่วยลดความเสี่ยงจากการจมน้ำ, การลื่นล้ม
